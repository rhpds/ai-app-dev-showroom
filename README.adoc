= AI App Development from Desktop to Production

A hands-on workshop that demonstrates how Red Hat addresses the challenges of moving AI applications from desktop experimentation to production on OpenShift. Participants play the role of a developer at Parasol Insurance, using Red Hat Developer Hub, the Red Hat build of Podman Desktop with AI Lab, OpenShift Pipelines, and OpenShift GitOps to build, test, and deploy an AI-powered chatbot.

**Rendered workshop**: https://rhpds.github.io/ai-app-dev-showroom

== Workshop Modules

* **Module 1: Provisioning and exploration** -- Self-service a developer workstation from RHDH, explore Podman Desktop and AI Lab
* **Module 2: Develop and build** -- Clone, customize, build, and deploy the Parasol Chat app
* **Module 3: Deploy and verify** -- Watch the CI/CD pipeline and GitOps deployment, verify the live app
